## Hook Test Plan

Following the Larry methodology from Oliver Henry's £670/mo case study, each slideshow tests ONE hook variable so we can iterate based on data, not guesses.

### Slideshow 1: PT Pre-Workout Swap (K&L)
**Hook Variable:** Authority figure reveal hook  
**Hook:** "My PT said replace pre-workout with this £3.50 bottle"  
**Why:** Authority endorsement (personal trainer) + practical swap + price anchor  
**Test:** Does professional authority drive more trust/conversions than peer recommendations?

### Slideshow 2: What's Actually In This Bottle (K&L)  
**Hook Variable:** Transparency/curiosity hook  
**Hook:** "What's actually in your £3.50 juice?"  
**Why:** Ingredient breakdown appeals to health-conscious audience who want to know what they're buying  
**Test:** Does radical transparency (weights, math) build more trust than benefit claims?

### Slideshow 3: Iron Deficient Authority (K&L)
**Hook Variable:** Medical authority + personal problem  
**Hook:** "My nutritionist told me to drink this every morning"  
**Why:** Higher medical authority + relatable health problem (iron deficiency common in UK women)  
**Test:** Does medical authority + personal struggle outperform general wellness claims?

### Slideshow 4: London Girlies Fridge (Skin Boost)
**Hook Variable:** Aspirational identity  
**Hook:** "London girlies — what's in my fridge instead of skincare"  
**Why:** Identity-based hook targeting London wellness demographic, beauty angle vs health  
**Test:** Does aspirational identity ("London girlies") + beauty benefits beat functional health claims?

### Slideshow 5: Boomer-Bait Mistake (K&L)
**Hook Variable:** Intentional imperfection for engagement  
**Hook:** "introducting Vita Press cold-pressed juice" [sic]  
**Why:** Typo on slide 1 drives comments ("it's introducing not introducting"), another typo on slide 3 ("pasterization" not pasteurization). Per Larry's "oven disappears" lesson — mistakes drive engagement.  
**Test:** Does comment-bait imperfection drive algorithm boost via high comment rate?

## What Metrics to Watch

### First 30 Minutes (Initial Algorithm Test)
- **Views:** Should hit 100-200 in first 30 min if algorithm picks it up
- **Watch time %:** TikTok shows this in analytics — aim for >50%
- **Shares:** High shares = algorithm boost

### 24-Hour Mark (Performance Assessment)
- **Views:** 
  - <500 = KILL (don't repost this hook)
  - 500-2K = OK (worth one more variation)
  - 2K-5K = GOOD (keep in rotation)
  - >5K = WINNER (make 3 variations immediately)
  
- **Comments:** 
  - Slideshow 5 should have highest comment rate (typo engagement)
  - Look for: "🧃" mentions, "where can I buy", "introducing*"
  
- **Profile visits:** High = hook is working, CTA is clear
  
- **DMs with 🧃:** Direct conversion intent

Posts typically peak at 24-48 hours, so check again at 72h for final view count and engagement rate.

## Larry Decision Rules

### After 24 Hours, For Each Slideshow:

**<500 views:**
- **Kill it.** Don't waste time on low-performing hooks.
- Try a radically different angle.

**500-2K views:**
- **Test one variation.** Tweak the hook slightly, keep the format.
- If variation also underperforms, drop the hook type.

**2K-5K views:**
- **Keep in rotation.** Solid performance.
- Test 1-2 variations over next week.

**>5K views:**
- **DOUBLE DOWN IMMEDIATELY.**
- Make 3 variations of the winning hook TODAY.
- Test different posting times (7:30am, 12pm, 4:30pm, 9pm).
- This is what works — milk it.

### Comment Patterns to Watch

**Slideshow 5 specifically:**
- Count comments mentioning the typo ("*introducing")
- High comment rate = algorithm boost even if views are moderate
- If comments >5% of views AND typos drive engagement, create more intentional-mistake slideshows

**All slideshows:**
- "Where can I buy?" = CTA is working
- "🧃" emoji usage = they're engaging with the CTA
- "What's in [blend]?" = transparency hooks working
- Negative comments = ignore, engagement is engagement

## Red Flags / When to Pivot

### If ALL 5 slideshows <500 views by 24h:
- **Account warmup issue:** TikTok flagged as bot/spam
  - Solution: Go back to warmup phase (scroll, like, comment for 7 days)
  
- **Content doesn't match audience:** Wrong niche targeting
  - Solution: Research UK wellness TikTok — what's actually working NOW?
  
- **No music added:** Silent slideshows get buried
  - Solution: Make sure every post has trending sound

### If High Views but Zero 🧃 Engagement:
- **CTA is broken:** They watch but don't know what to do
  - Solution: Test different CTAs:
    - "Comment 🧃 to pre-order"
    - "DM 🧃 for Sunday pickup"
    - "Link in bio for pre-order"
  
- **Price is off:** £3.50 feels too expensive/cheap
  - Solution: Test £3 / 3 for £8.50 or £4 / 3 for £10

### If High Views + High 🧃 but Zero Actual Orders:
- **Fulfillment concern:** People don't trust Sunday pickup/delivery
  - Solution: Show behind-the-scenes of actual market stall, real customers
  
- **Payment friction:** No clear payment method
  - Solution: Set up simple payment link, mention in response to 🧃 comments

**Don't post blindly for a month then check analytics.** Check EVERY DAY. Adjust EVERY DAY.
