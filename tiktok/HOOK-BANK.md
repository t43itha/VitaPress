# Vita Press TikTok Hook Bank — 50 Hooks for Iteration Ammunition

## How to use this bank

When a launch slideshow gets **>5K views in 24h**, immediately ship 3–5 variants from this bank using the **same archetype** but different specifics (different SKU, different authority figure, different identity group). Don't reinvent the hook — clone what worked and rotate one variable.

When a slideshow **dies <500 views**, mark that archetype as cold for **2 weeks** and pull from the other four buckets. Don't keep watering a dead hook.

**Iteration cadence:** 1–2 posts per day, every day. Hooks are cheap. Volume is the moat.

**Format on every slide:** white text + thick black stroke, centered ~28% from top. Last slide always = bottle hero + CTA `Comment 🧃 to pre-order — Sunday pickup or local London delivery`.

**Variable tested** = the ONE thing different from the parent slideshow. Never test two variables at once.

## Iteration playbook

**Day 1–5:** Ship the 5 launch slideshows. Track first 30-min views in the Notion tracker.

**Day 6 onwards:**
- Any slideshow with >5K @ 24h → pull 3 variants from the SAME archetype bucket, ship within 48h
- Any slideshow with <500 @ 24h → mark that archetype cold for 2 weeks, pull from another bucket
- Default cadence: 2 posts/day, 12pm and 6pm UK
- Monday + Tuesday lunch slot is highest baseline reach for UK wellness

**Comment reply rule:** reply to first 20 comments on every post within 30 min of going live. This is non-negotiable — TikTok algorithm reads it as quality signal.

**Profile bio rotation:** every Monday, swap the bio link CTA to whichever SKU is winning that week.

**Kill conditions:** if 7 consecutive posts get <500 views, stop posting for 72h, audit the audience signal in TikTok analytics, then resume with a fresh archetype.
